# 根据未拆分完成的总表文件进行筛选并添值至已拆分的表格文件

## 用途（规划）

在「总表尚未按门店完全拆分」或「总表与已拆分结果需要对齐」的场景下：

- 用**总表**（或未完成拆分的汇总表）做条件筛选；
- 将筛选得到的字段或计算结果**写回**已经按门店拆分好的各 `.xlsx` 文件中（按门店/主键等规则对齐）。

具体列名、匹配键、写入策略待与业务确认后在 `Filter_add.py` 中实现。
