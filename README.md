## 项目目录说明

### `按门店维度拆分表格/`

用于将 **单个 Excel（`.xlsx`）** 按表中的 **店铺（或门店）等某一列** 拆成 **多个独立表格文件**：每个不同取值对应一个 `.xlsx`，方便按门店分发、核对或归档。
