# 按门店维度拆分表格

将单个 Excel（`.xlsx`）按某一列（通常为「店铺 / 门店」）拆成多个文件：每个不同取值对应一个独立的 `.xlsx`，便于按门店分发或归档。

## 功能说明

- 图形界面（Qt / PySide6）：选择输入文件、输出文件夹、用于拆分的列名；支持常见表头自动识别（如「店铺」「门店」「Store」等）。
- 输出文件名由该列单元格内容生成，已做 Windows 非法字符处理；同名会自动加 `_2`、`_3` 后缀。
- 可选命令行模式，便于脚本调用。

## 环境要求

- Windows 10/11（亦可在其他系统运行源码，打包脚本以 Windows 为主）。
- Python 3.10+（若从源码运行）。

## 安装依赖（源码运行 / 打包前）

在**本目录**打开终端，执行：

```bash
pip install -r requirements.txt
```

主要依赖：`pandas`、`openpyxl`、`PySide6`；打包 exe 还需 `pyinstaller`（已写在 `requirements.txt` 中）。

## 图形界面使用步骤

1. 运行主程序：
   ```bash
   python Tabellen_teilen.py
   ```
2. 点击 **「浏览」**（Excel 文件）：选择要拆分的 `.xlsx`。
3. 程序会读取表头并填充 **「店铺列」** 下拉框；若识别到常见列名会自动选中，否则请手动选择用于分组的列。
4. 点击 **「浏览」**（输出目录）：选择保存拆分结果的文件夹。
5. 点击 **「开始拆分」**；完成后会提示生成的文件个数。大表处理时界面可能短暂显示「正在拆分，请稍候…」，请勿重复点击。

## 命令行模式（无界面）

```bash
python Tabellen_teilen.py --cli "输入文件.xlsx" "店铺列名" "输出目录路径"
```

- 第 4 个参数（输出目录）可省略，默认在输入文件同目录下创建 `拆分结果` 文件夹。
- 「店铺列名」须与 Excel 表头**完全一致**（含空格时建议先在 Excel 里确认表头文本）。

## 打包为 exe（Windows）

双击或在命令行运行本目录下的 **`build_exe.bat`**。完成后在 `dist` 目录可得到 **`按门店拆分表格.exe`**（单文件、无控制台窗口）。

也可使用已生成的 `按门店拆分表格.spec`：

```bash
pyinstaller 按门店拆分表格.spec
```

说明：`dist/`、`build/` 体积较大，已通过仓库根目录 `.gitignore` 忽略，不要提交到 Git。

## 常见问题

- **提示所选列不存在**：重新选择 Excel 后，在下拉框中选对与表头一致的列名。
- **表格为空**：源文件无数据行时无法拆分。
- **拆分很慢**：大文件会完整读入内存并逐组写出，属正常现象；可关闭其他占用内存的程序。
